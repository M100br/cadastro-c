# Sistema de Cadastro de Alunos em C
  Projeto acadêmico desenvolvido para praticar lógica de programação na linguagem C.
## Funcionalidades
- Cadastro de alunos
- Armazenamento de dados em memória
- Exibição das informações cadastradas
- Menu interativo no terminal

## Tecnologias utilizadas
- Linguagem C

## Contexto acadêmico
Projeto desenvolvido no 2º semestre do curso de Análise e Desenvolvimento de Sistemas  
FATEC Mogi das Cruzes.
